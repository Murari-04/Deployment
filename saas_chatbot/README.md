# 🤖 AI SaaS Chatbot

An AI-powered SaaS chatbot built using **FastAPI**, **Google Gemini API**, and a responsive **HTML, CSS, and JavaScript** frontend. The chatbot provides intelligent real-time conversations through a clean and user-friendly interface.

---

## 🚀 Features

- 💬 AI-powered conversational chatbot
- 🤖 Google Gemini AI integration
- ⚡ FastAPI backend
- 🌐 Responsive HTML, CSS & JavaScript frontend
- 🔒 Secure API key management using environment variables
- ☁️ Deployed on Vercel
- 🔄 REST API architecture

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Deployment.git
```

### Navigate to the project

```bash
cd Deployment/saas_chatbot
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` file

```env
GEMINI_API_KEY=your_api_key_here
```

### Run the FastAPI server

```bash
uvicorn backend.index:app --reload
```

Open:

```
http://127.0.0.1:8000
```
